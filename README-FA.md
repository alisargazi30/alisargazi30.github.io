# 🚀 علیرضا سرگزی - وبسایت رزومه شخصی

یک وبسایت رزومه شخصی جهانی و دوزبانه که با Next.js 15 ساخته شده و دارای انیمیشن‌های فوق‌العاده، یکپارچگی با هوش مصنوعی، و سیستم مدیریت محتوای جامع است.

## ✨ ویژگی‌ها

### 🎨 طراحی و تجربه کاربری
- **انیمیشن‌های خیره‌کننده**: انیمیشن‌های پیشرفته Framer Motion با افکت‌های مغناطیسی، سیستم ذرات، و انتقال‌های نرم
- **پشتیبانی دوزبانه**: پشتیبانی کامل از فارسی (راست به چپ) و انگلیسی (چپ به راست) با تایپوگرافی مناسب
- **تم تاریک/روشن**: تغییر تم بدون مشکل با انتقال‌های نرم
- **طراحی ریسپانسیو**: رویکرد موبایل اول با تطبیق کامل در تمام دستگاه‌ها
- **افکت‌های شیشه‌ای**: رابط کاربری مدرن با تاری پس‌زمینه و شفافیت
- **کرسر سفارشی**: کرسر تعاملی با افکت‌های درخشش و جذب مغناطیسی

### 🛠 برتری فنی
- **Next.js 15**: آخرین App Router با کامپوننت‌های سرور و استریمینگ
- **TypeScript**: ایمنی نوع کامل در سراسر برنامه
- **Tailwind CSS v4**: استایل‌دهی مدرن با سیستم طراحی سفارشی
- **Framer Motion**: انیمیشن‌های پیشرفته و تعامل‌های ریز
- **یکپارچگی Supabase**: دیتابیس فول‌استک با امنیت سطح ردیف
- **فونت‌های فارسی**: فونت وزیرمتن برای تایپوگرافی زیبای فارسی

### 📱 بخش‌های اصلی
- **بخش قهرمان**: معرفی انیمیشنی با چرخش نقش و افکت‌های ذرات
- **درباره من**: پیشینه حرفه‌ای با آمار انیمیشنی
- **مهارت‌ها**: نوارهای مهارت تعاملی با انیمیشن‌های پیشرفت
- **نمونه کارها**: نمایش پروژه‌های پویا با قابلیت فیلتر
- **وبلاگ**: سیستم وبلاگ کامل با پنل ادمین
- **تماس**: فرم‌های درخواست مشاوره و آموزش

### 🔐 ویژگی‌های ادمین
- **احراز هویت**: سیستم ورود امن با Supabase Auth
- **مدیریت وبلاگ**: ایجاد، ویرایش و انتشار مقالات به دو زبان
- **مدیریت درخواست‌ها**: مدیریت درخواست‌های مشاوره و آموزش
- **داشبورد**: آمار لحظه‌ای و دسترسی سریع به تمام ویژگی‌ها

### 🤖 هوش مصنوعی و اتوماسیون
- **یکپارچگی چت‌بات**: آماده برای پیاده‌سازی چت‌بات هوش مصنوعی
- **اتوماسیون n8n**: قابلیت‌های اتوماسیون گردش کار
- **مشاوره هوش مصنوعی**: خدمات تخصصی هوش مصنوعی و یادگیری ماشین

## 🚀 شروع کار

### پیش‌نیازها
- Node.js 18+ 
- npm یا yarn
- حساب Supabase

### نصب

1. **کلون کردن مخزن**
   \`\`\`bash
   git clone https://github.com/alirezasargazi/portfolio.git
   cd portfolio
   \`\`\`

2. **نصب وابستگی‌ها**
   \`\`\`bash
   npm install
   \`\`\`

3. **تنظیم متغیرهای محیطی**
   \`\`\`bash
   cp .env.example .env.local
   \`\`\`
   
   اطلاعات Supabase خود را وارد کنید:
   \`\`\`env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
   NEXT_PUBLIC_DEV_SUPABASE_REDIRECT_URL=http://localhost:3000
   \`\`\`

4. **راه‌اندازی دیتابیس**
   \`\`\`bash
   # اجرای اسکریپت ساختار دیتابیس
   npm run db:setup
   
   # بارگذاری داده‌های اولیه
   npm run db:seed
   \`\`\`

5. **شروع سرور توسعه**
   \`\`\`bash
   npm run dev
   \`\`\`

6. **باز کردن مرورگر**
   به آدرس `http://localhost:3000` بروید

├── app/                    # Next.js App Router

│   ├── auth/              # صفحات احراز هویت
│   └── api/               # مسیرهای API
├── components/            # کامپوننت‌های قابل استفاده مجدد
│   ├── ui/               # کامپوننت‌های پایه UI
│   └── ...               # کامپوننت‌های ویژگی
├── lib/                  # ابزارها و پیکربندی‌ها
│   ├── i18n/            # بین‌المللی‌سازی
│   ├── supabase/        # کلاینت دیتابیس
│   └── utils.ts         # توابع کمکی
├── scripts/             # اسکریپت‌های دیتابیس
└── public/              # دارایی‌های استاتیک
\`\`\`

## 🗄️ ساختار دیتابیس

### جداول
- **blog_posts**: مقالات وبلاگ دوزبانه با متادیتا
- **consultation_requests**: درخواست‌های مشاوره مشتریان
- **training_requests**: درخواست‌های آموزش و دوره
- **portfolio_projects**: داده‌های نمایش پروژه

### امنیت
- امنیت سطح ردیف (RLS) فعال روی تمام جداول
- دسترسی فقط ادمین برای مدیریت محتوا
- دسترسی خواندن عمومی برای محتوای منتشر شده

## 🌐 بین‌المللی‌سازی

### زبان‌های پشتیبانی شده
- **انگلیسی (en)**: چیدمان چپ به راست
- **فارسی (fa)**: چیدمان راست به چپ با تایپوگرافی مناسب

### اضافه کردن زبان‌های جدید
1. فایل دیکشنری در `lib/i18n/dictionaries/` ایجاد کنید
2. زبان را به `lib/i18n/config.ts` اضافه کنید
3. در صورت نیاز پیکربندی فونت‌ها را به‌روزرسانی کنید

## 🎨 سفارشی‌سازی

### رنگ‌های تم
پالت رنگ را در `app/globals.css` تغییر دهید:
\`\`\`css
:root {
  --primary: your-color;
  --secondary: your-color;
  /* ... */
}
\`\`\`

### انیمیشن‌ها
انیمیشن‌ها را در فایل‌های کامپوننت سفارشی کنید یا جدیدهایی به `globals.css` اضافه کنید

### محتوا
اطلاعات شخصی را در موارد زیر به‌روزرسانی کنید:
- فایل‌های دیکشنری (`lib/i18n/dictionaries/`)
- اسکریپت بذر دیتابیس (`scripts/002_seed_portfolio_data.sql`)
